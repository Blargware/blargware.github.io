April 8, 2008 - The Death of DraftCap - MTGO v3.0 is apparently due to come out in the next week or so. (See here.) This makes me happy because I'd love to see an MTGO that Wizards can really push without worrying about stability. However, it also makes me sad because it will mean the end of the line for DraftCap. (Reason: Much of the 'text recognition' that's built into DraftCap works by by simple brute-force matching of characters. This worked because MTGOv2 only had four different 'font sizes,' at least in the drafting area. MTGOv3 instead has scalable fonts, and throws in anti-aliasing just to make it more difficult.) The amount of work that would be required to update DraftCap for MTGOv3 just isn't worth it for me.

In the past two years I've really enjoyed watching the effect DraftCap has had on the discussion of drafting, and I'm sure it's helped a great many players (myself included) to improve by looking over the shoulder of those better than us. I sincerely hope that WotC recognizes the value that such discussion brings to the card table and puts in the effort to make MTGOv3 log drafts on its own. From here on out it's up to them. Thanks to those people who appreciated DraftCap enough to donate, or just to write and say thanks. It helped to justify the scores of hours that got put into this. Hopefully someday I'll find time for another Magic-related project that'll be even more exciting.

March 24, 2008 - Version 0.947 of MTGO DraftCap has been released. This fixes a minor bug with land images for Lorwyn.

March 6, 2008 - Version 0.946 of MTGO DraftCap has been released. This has beta Morningtide support.

November 11, 2007 - Version 0.945 of MTGO DraftCap has been released. This fixes planeswalkers and card images for cards with hyphens in them.

November 3, 2007 - Version 0.944 of MTGO DraftCap has been released. This has BETA Lorwyn support. Planeswalker borders appear to be different, so they might not work. Please turn on 'Send Error Emails' in preferences to make sure any bugs come to me.

September 17, 2007 - Version 0.943 of MTGO DraftCap has been released. This fixes black card borders in HTML output, adds a preference for disabling borders, and supports drafting of Masters Edition. NOTE: Because Masters Edition uses many old sets, the data files are not being included by default. To draft Masters Edition, please go to the DraftCap page and download/install the appropriate data files.

August 31, 2007 - Version 0.942 of MTGO DraftCap has been released. It's like 0.941, except that I remembered to include all of the 10th Edition files this time.

August 24, 2007 - Version 0.941 of MTGO DraftCap has been released. In addition to adding 10th edition support, I've added black borders to the card images in HTML output (to look like actual magic cards). Side note: It's entirely possible for people to 'update' DraftCap themselves to handle new sets - it simply requires updates to various text files in the data directory.

June 30, 2007 - Version 0.940 seems to work fine for Future Sight. Also, Joseph T. emailed me to let me know that DraftCap works fine on Vista, but you need to right-click on it and run it as administrator (which is apparently the same thing you need to do to run MTGO itself). I can't verify this since I don't have access to Vista, but hopefully this will be useful to some people.

June 4, 2007 - Version 0.940 of MTGO DraftCap has been released, with BETA Future Sight support. There were NO code changes this time, only updates to the various text data files. As a programmer I find that pretty awesome. (Note: There may still be bugs :)

March 13, 2007 - Version 0.939 of MTGO DraftCap has been released, fixing the problems mentioned in previous post.

March 7, 2007 - There are a couple current bugs that I'm aware of: Decklists are frequently not showing up, packs in which Dead/Gone is the first card don't show anything else, and Frozen Aether causes some issues. If there are others I've not mentioned, please let me know. I'll try to have these fixed sometime in the coming week. Also, if anyone could verify that DraftCap works for them on Windows Vista I'd much appreciate it.

February 25, 2007 - Version 0.938 of MTGO DraftCap has been released. This has BETA Planar Chaos support. As Planar Chaos-specific bugs are discovered I will try to fix them ASAP. If you'd like to help, please open the preferences (via right-click on icon) and turn on 'Send Error E-mails.'

December 7, 2006 - Version 0.937 of MTGO DraftCap has been released. This should hopefully be the last Time Spiral release. This set has really put DraftCap to the test, due to the multiple cardfaces in the same pack and the mixed card titles. DraftCap now does a lot more guesswork and prediction when dealing with hard-to-parse cards, so the odds of mislabeling have gone up significantly. If this happens to you, PLEASE e-mail me immediately.

November 19, 2006 - Version 0.936 of MTGO DraftCap has been released. This should fix all issues that have been seen since Time Spiral was released.

October 31, 2006 - Version 0.935 of MTGO DraftCap has been released. This has BETA Time Spiral support. Since WotC no longer does drafts on beta, I'm mostly just guessing as to how well this will actually work - I've run it through some fabricated test cases, but haven't used it on a real draft. Also, this set seems like it was designed to make updating DraftCap difficult - between having mixed card frames, an old-style split card, and a card with entirely italics font, Time Spiral was a little more headache-inducing than most sets.

September 2, 2006 - Version 0.934 of MTGO DraftCap has been released. Fixed some more Coldsnap-brought bugs.

August 22, 2006 - Version 0.933 of MTGO DraftCap has been released. Fixed some bugs brought on by Coldsnap release.

August 17, 2006 - Version 0.932 of MTGO DraftCap has been released - Coldsnap images fixed. For real this time.

July 17, 2006 - Version 0.931 of MTGO DraftCap has been released - Coldsnap images fixed. EDIT: Apparently not. They would've been fixed had there not been a different bug I never discovered. The easy fix is to go to MTGO_DraftCap\data\cardlists directory and rename the file 'coldsnap.txt' to 'csp.txt'. NOTE: If you do this, you'll need to fix it or uninstall before you install a new version.

July 16, 2006 - Version 0.930 of MTGO DraftCap has been released, with Coldsnap support and new integrated downloading capability. Unfortunately Coldsnap images don't show up correctly.

June 27, 2006 - Rich Hoaen appears to be starting a series of daily draftcaps on SCG. Hopefully this will prove useful to a lot of people.
